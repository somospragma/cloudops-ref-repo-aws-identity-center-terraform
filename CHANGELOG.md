## [1.0.0] - 2026-04-28

### Added
- Initial release of IAM Identity Center governance module
- Support for creating users via JSON configuration
- Support for creating groups via JSON configuration
- Support for creating permission sets with managed and inline policies
- Support for customer managed policies attachment
- Support for permissions boundary configuration
- Group to account assignments with permission sets
- User to group memberships
- Direct user to account assignments (optional)
- Tags support for all resources
